Learn The Fundamentals

You must have the mindset that every developer must have. This mindset will help you make better programming decisions.

First of all, you must understand the purpose of the software. In fact, there is one purpose of all software: helping people. Remember, the purpose of software is not to show you how smart you are. Developers who cannot understand the purpose of software will write bad software. What is bad software? A complex system that does not help people very much. When making software decisions, you should be guided by yourself, always keeping this in mind: how can I help?
